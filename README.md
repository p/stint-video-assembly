## File Format

The skeleton file has the following form:

    SCHD0768.MOV:
      split: 
      join: 
      size: 2438 MB

`split` and `join` fields determine what to do with the file.
`size` field is informational.

Single split:

    SCHD0768.MOV:
      split: s1-oleg
      join: 
      size: 2438 MB
